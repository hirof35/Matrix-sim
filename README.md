# 🧮 万能行列計算・可視化シミュレータ (Matrix Visual Simulator)

Rust、`eframe (egui)`、および `nalgebra` を用いて構築された、リアルタイムの行列計算および2次元線形変換の可視化シミュレータです。
数式上の数値が、幾何学的に空間をどのように歪ませるのかを直感的に理解することができます。

## 🌟 主な機能

- **マルチサイズ対応**: 2x2、3x3、4x4 の正方行列の入力・計算に対応。
- **2つの計算モード**:
  1. **単一行列Aの特性**: 行列式 ($\det A$) の自動計算、および逆行列 ($A^{-1}$) のリアルタイム算出。
  2. **2つの行列の積 ($A \times B$)**: 行列の乗算結果と、合成行列の行列式の検証。
- **2D線形変換のリアルタイム可視化 (※2x2モード限定)**:
  - 行列の数値変更に合わせて、**空間の格子線（緑）** と **単位円（青）** がどう変形（回転・拡大縮小・せん断・潰れ）するかをグラフィカルに描画。
- **日本語対応**: OS内蔵のフォントを利用した文字化け（豆腐文字）防止ロジックを搭載。
